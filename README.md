### Option 2: From source

Requires [Node.js](https://nodejs.org) and [Git](https://git-scm.com).

```bash
git clone https://github.com/tabblar/tab-organizer.git
cd tab-organizer
npm install
npm start
```

## ✨ Features

- Group related tabs into organized workspaces
- Save and restore tab sessions instantly
- Declutter your browser and stay focused
